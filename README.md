This project was an optional activity from Algoritmos (Algorithms) in college, and the five calculators developed here were actually six out of eight of the activities - the remaining two being string input and output, so it was too basic to fit alongside these calculators.

I believe the structure of this code is closer to that of a C code, in terms of the use of functions. But the expected result was achieved either way, with the hardest part being enabling the scanner to pass from function to function.
